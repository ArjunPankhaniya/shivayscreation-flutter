# 🛍️ ShivaysCreation Flutter  
A Modern E-commerce App Built with Flutter & Firebase  

ShivaysCreation Flutter is a full-featured e-commerce application designed for a seamless online shopping experience.  
This is the **Flutter-based successor** to the original **Java-based Android app**, now re-engineered for:  
🚀 **Cross-Platform Compatibility** (Android & iOS)  
⚡ **Improved Performance & Scalability**  
🎨 **Enhanced UI/UX**  

---

## ✨ Features & Modules  

### 🔐 **Authentication & User Management**  
✅ **Firebase Authentication** – Secure login & signup using email/password  
✅ **Google Authentication** – Quick sign-in with Google  
✅ **User Profiles** – Manage personal details, order history, and preferences  

### 🛒 **Product & Order Management**  
✅ **Dynamic Product Listings** – Fetch real-time product data from Firebase Firestore  
✅ **Product Categories & Search** – Easily browse and search for products  
✅ **Wishlist & Cart System** – Add/remove items with smooth UI transitions  
✅ **Order Placement & Tracking** – Store order history and track deliveries  

### 💳 **Secure Payments & Transactions**  
✅ **Razorpay Payment Gateway** – Seamless UPI, card, and net banking payments  
✅ **Order Confirmation & Receipts** – Automatic invoice generation  

### 📢 **Push Notifications & Engagement**  
✅ **Firebase Cloud Messaging (FCM)** – Real-time push notifications for offers & updates  
✅ **Flutter Toast Messages** – User-friendly alerts & confirmations  

### 📷 **Media & UI Enhancements**  
✅ **Firebase Storage Integration** – Upload & display product images efficiently  
✅ **Cached Network Images** – Faster loading and reduced bandwidth usage  
✅ **Animated UI** – Smooth transitions with `flutter_staggered_animations`  

### 📊 **Admin Panel Integration (Future Scope)**  
✅ **Product Upload & Management** – Admins can add/update/delete products  
✅ **Order Management** – View and process user orders  
✅ **Sales Analytics Dashboard** – Track revenue, orders, and user activity

## 🚀 How to Run the Project?

### 1️⃣ Clone the Repository  
```sh
git clone https://github.com/your-username/shivayscreation-flutter.git
cd shivayscreation-flutter
```

### 2️⃣ Install Dependencies  
```sh
flutter clean
flutter pub get
```

### 3️⃣ Configure Firebase  
- Create a Firebase project in [Firebase Console](https://console.firebase.google.com/).  
- Download `google-services.json` and place it inside `/android/app/`.  
- Download `GoogleService-Info.plist` (for iOS) and place it inside `/ios/Runner/`.  

### 4️⃣ Run the App  
```sh
flutter run
```
(Ensure you have an emulator or a connected physical device.)
